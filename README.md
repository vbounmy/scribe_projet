# Scribe
Scribe est un outil en ligne de commande qui transforme un enregistrement 
audio en compte rendu écrit.

## Fonctionnement
Scribe fonctionne en trois temps. D'abord, on fournit un fichier audio. Ensuite, un modèle de transcription (STT) le convertit en texte brut. Enfin, un LLM reformule ce texte en compte rendu structuré (titre, points clés, décisions, actions), qui est sauvegardé dans le dossier de sortie.
